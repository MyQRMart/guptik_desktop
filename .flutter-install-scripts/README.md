# GupTik Desktop - Cross-Platform Installation

The app is ready for **Linux**, **macOS**, and **Windows**.

## Installation

Run from the project root directory:

### Linux & macOS
```bash
bash install.sh
```

### Windows (PowerShell)
```powershell
powershell -ExecutionPolicy Bypass -File install.sh
```
Or simply:
```powershell
.\.flutter-install-scripts\windows-install.ps1
```

## Uninstallation

### Linux & macOS
```bash
bash uninstall.sh
```

### Windows (PowerShell)
```powershell
powershell -ExecutionPolicy Bypass -File uninstall.sh
```
Or:
```powershell
.\.flutter-install-scripts\windows-uninstall.ps1
```
